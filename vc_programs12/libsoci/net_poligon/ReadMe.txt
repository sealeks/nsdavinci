========================================================================
КОНСОЛЬНОЕ ПРИЛОЖЕНИЕ. Общие сведения net_poligon
========================================================================

Это приложение net_poligon создано автоматически с помощью 
мастера приложений.

   В этом файле представлена сводка содержимого всех файлов, входящих в 
   состав приложения net_poligon.


net_poligon.vcproj
   Основной файл проекта VC++, автоматически создаваемый с помощью 
   мастера приложений.
   В файле представлены сведения о версии Visual C++, использованной 
   при создании файла, а также о параметрах платформы, конфигурации и 
   проекта, заданных с помощью мастера приложений.

net_poligon.cpp
   Основной исходный файл приложения.

/////////////////////////////////////////////////////////////////////////////
Другие стандартные файлы:

StdAfx.h, StdAfx.cpp
   Эти файлы используются для построения файла предкомпилируемых заголовков 
   (PCH) net_poligon.pch и файла предкомпилируемых типов 
   StdAfx.obj.
